# 任务目录

此目录用于存放 Addax 任务的 JSON 配置文件。

## 说明

后端服务运行时会使用此目录存储和读取任务配置文件。在 Docker 部署时，此目录会被挂载为容器卷，使得：

1. 任务配置文件在容器重启后依然保留
2. 可以在宿主机直接查看和管理任务文件
3. 方便备份和迁移

## 目录结构

任务文件会按照一定规则存储在此目录下，由后端服务自动管理。

## Docker 挂载

在 `docker-compose.yml` 中已配置：

```yaml
volumes:
  - ./backend/job:/app/job
```

这样容器内的 `/app/job` 目录会映射到宿主机的 `backend/job` 目录。
